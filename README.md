# Student Management System

A web-based **Student Management System** developed using **Python Flask and MySQL**.
This project allows users to manage student records through a simple and user-friendly interface.

## Features

* User Registration and Login System
* Add New Students
* View Student Details
* Edit Student Information
* Delete Student Records
* Database Integration with MySQL

## Technologies Used

* Python
* Flask
* MySQL
* HTML
* CSS
* Bootstrap
